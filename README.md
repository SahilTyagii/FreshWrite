<img src="public/images/Home.gif" alt = "Home Page">

### Project Setup

1. **Clone the Repository**
   ```
   git clone https://github.com/SahilTyagii/FreshWrite.git
   ```
2. **Install Dependencies**
   ```
   npm i
   ```
3. **Start Development Server**
   ```
   nodemon server.js
   ```
